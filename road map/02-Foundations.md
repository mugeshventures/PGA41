# 02. Foundations

## Core concepts every AI Engineer should know

### 1. Programming and Computer Science
- Python as the primary language
- Data structures, algorithms, and software design
- Version control with Git and GitHub

### 2. Mathematics and Statistics
- Linear algebra: vectors, matrices, eigenvalues
- Probability and statistics: distributions, sampling, confidence
- Calculus basics for optimization

### 3. Data Engineering
- Data cleaning, transformation, and pipelines
- SQL and NoSQL fundamentals
- Working with CSV, JSON, parquet, and streaming data

### 4. Machine Learning Basics
- Supervised vs unsupervised learning
- Model evaluation: accuracy, precision, recall, F1
- Overfitting, regularization, and cross-validation

### 5. AI and model fundamentals
- Pre-trained models and transfer learning
- Model inference vs training
- Feature engineering and embeddings

## Recommended learning path

1. Build small projects using datasets like Iris, MNIST, and tabular data.
2. Practice data pipelines and experiment tracking.
3. Learn how to choose the right model for different problem types.
