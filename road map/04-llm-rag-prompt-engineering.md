# 04. LLMs, RAG, Prompt Engineering

## Large Language Models (LLMs)

### What to learn
- How LLMs work: attention, tokens, context windows
- Differences between OpenAI GPT, Anthropic Claude, Google Gemini, Meta Llama, Mistral, Cohere
- When to use closed-source vs open-source models

## Prompt Engineering

### Core skills
- Writing effective prompts for few-shot, zero-shot, and one-shot scenarios
- Using chain-of-thought (CoT), reasoning prompts, and ReAct patterns
- Managing prompt size, token limits, and context windows

### Advanced prompt techniques
- Prompt templates and dynamic prompt generation
- Input/output constraints and guardrails
- Prompt testing, debugging, and adversarial prompt handling

## Retrieval Augmented Generation (RAG)

### What RAG means
- Combining external knowledge sources with an LLM
- Indexing documents or embeddings in a vector database
- Querying the index and feeding retrieved context into prompts

### Key components
- Embeddings generation and vector storage
- Vector DBs like Pinecone, Weaviate, Milvus, Redis, and Chroma
- Re-ranking and retrieval pipelines

## AI Agents

- Building simple agents that orchestrate multiple prompts or API calls
- Using tools and action plans inside an LLM workflow
- Chaining LLM calls for planning, retrieval, and execution
