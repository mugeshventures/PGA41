# 07. Projects and Career Path

## Build practical AI projects

### Project ideas
- Chatbot with retrieval-augmented generation (RAG)
- Document search using embeddings and a vector database
- AI agent that performs multi-step tasks
- End-to-end application integrating AI into a product

### Learning goals
- Practice end-to-end pipelines from data ingestion to deployment
- Demonstrate model integration, monitoring, and user feedback loops
- Show real-world value with metrics and usability

## Career-focused milestones

- Create a portfolio of AI applications and demos
- Share projects on GitHub and technical blogs
- Participate in AI communities and roadmap groups
- Learn role responsibilities across product, engineering, and research teams

## Continuous growth

- Stay current with LLM and AI tool releases
- Explore new techniques in prompt engineering, agents, and model tuning
- Learn emerging tools like Ollama, LM Studio, and next-generation LLM SDKs
