# 03. Models and Training

## Understanding model categories

### Pre-trained models
- Use models from Hugging Face, OpenAI, Anthropic, Google Gemini, Meta Llama, Mistral, Cohere
- Focus on fine-tuning, adapters, and prompt-based adaptation

### Neural networks and architectures
- Feedforward networks, CNNs, RNNs, and transformers
- Transformers as the backbone for modern LLMs and vision models

### Training vs inference
- Training: model updates from large datasets
- Inference: using a model to generate predictions or outputs

## Practical AI Engineer skills

- Fine-tuning and transfer learning
- Model evaluation and validation on real-world test sets
- Hyperparameter tuning and efficiency improvements
- Performance monitoring during inference

## Tools and platforms

- Hugging Face Hub and Inference API
- TensorFlow, PyTorch, and ONNX
- MLflow, BentoML, and Weights & Biases
